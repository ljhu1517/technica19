# About Dataset

Forest area of countries from years 1990 - 2015

### Data columns

| Field Name      | Description                                  |
| --------------- | -------------------------------------------- |
| country_name    | Country name                                 |
| country_code    | Country abbreviation code                    |
| year            | Year                                         |
| pct_forest_land | Percentage of land that is forest in country |
